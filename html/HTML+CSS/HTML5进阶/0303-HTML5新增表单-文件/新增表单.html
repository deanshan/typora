2、fromentype的三个参数

application/x-www-form-urlencoded 在发送前对所有字符进行编码（默认）。

multipart/form-data 不对字符编码。当使用有文件上传控件的表单时，该值是必需的。

text/plain 将空格转换为 "+" 符号，但不编码特殊字符。

3、formtarget:提交后展现方式，覆盖target属性（submit或image,_blank、_self、_parent…）

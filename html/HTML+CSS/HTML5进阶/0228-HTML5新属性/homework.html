1、html4和HTML5的不同之处，HTML5的优缺点，简述HTML5和H5的区别

html4和HTML5的不同之处：HTML5简化了语法(如doctype的声明)，HTML5新增了些标签、属性、接口并删除了部分标签、属性、接口

HTML5的优缺点：1、部分浏览器并不能完全兼容HTML5，造成用户体验不佳。 2、自从它诞生后，它的原则就是所有的技术必须是开放的，不准有专利限制，这方面也是引起争议的方面

HTML5和H5的区别：1、HTML5是定义 HTML 标准的最新的版本，2012年推出的HTML最新标准。 2、H5:是指HTML5+CSS3+JavaScript等技术的组合

# 技术栈
  + supervisor：代替node启动服务器，并实时监听
  + express：创建服务器
  + express-static：处理静态文件
  + body-parser：解析post数据 application/x-www-form-urlencoded
  + multer：处理post文件数据  multipart/form-data
  + cookie-parser：解析cookie
  + cookie-session：处理session
  + jade、ejs（两者选其一）
  + consolidate 适配模板引擎
  + mysql：数据库
